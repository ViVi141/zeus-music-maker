# 宙斯音乐制作器 - 系统要求

## 📋 最低系统要求

### 操作系统
- **最低版本**: Windows 10 版本 1607 (周年更新, Build 14393) 或更高
- **推荐版本**: Windows 11 (所有版本)
- **架构**: x64 (AMD64/Intel 64) - **仅支持 64 位系统**
- **服务器版本**: Windows Server 2016 或更高版本

### 硬件要求

#### 处理器
- **最低**: 支持 x64 指令集的处理器
- **推荐**: 多核心处理器 (4 核或更多)
- **说明**: 多核心处理器可以显著提升并行处理性能

#### 内存 (RAM)
- **最低**: 4GB RAM
- **推荐**: 8GB RAM 或更多
- **说明**: 
  - 基础运行需要约 50-100MB
  - 处理大文件时内存占用会动态增长
  - 视频处理需要更多内存

#### 存储空间
- **程序文件**: 约 8-10MB (单文件便携版)
- **FFmpeg**: 约 50-100MB (首次运行时自动下载)
- **临时文件**: 约 50-200MB (取决于处理文件大小)
- **配置文件**: 约 1-5MB (用户配置和项目数据)
- **总计最低**: 200MB 可用空间
- **推荐**: 500MB 或更多可用空间

#### 显示
- **最低分辨率**: 1024x768
- **推荐分辨率**: 1920x1080 或更高
- **DPI 支持**: 支持高 DPI 显示 (PerMonitorV2)
- **说明**: 程序会自动适配不同 DPI 设置

### 软件要求

#### 运行时库
- **Microsoft Visual C++ Redistributable**: 
  - 通常 Windows 10/11 已预装
  - 如缺失，可从 Microsoft 官网下载
- **Windows Common Controls 6.0**: 
  - Windows Vista+ 已包含
  - 无需单独安装

#### 网络连接
- **首次运行**: 需要网络连接以下载 FFmpeg
- **后续使用**: 可离线使用 (FFmpeg 已下载)
- **下载速度**: 推荐稳定的网络连接 (FFmpeg 约 50-100MB)

#### 字体支持
- **中文字体**: Windows 自带字体即可
- **推荐字体**: 
  - 微软雅黑 (msyh.ttc)
  - 黑体 (simhei.ttf)
  - 宋体 (simsun.ttc)
- **说明**: 程序会自动检测并使用系统字体

### 权限要求
- **管理员权限**: ❌ 不需要
- **文件权限**: ✅ 需要文件读写权限
  - 读取输入文件
  - 写入输出文件
  - 创建临时文件
  - 保存配置文件

## 🎯 推荐系统配置

### 操作系统
- **最佳**: Windows 11 (64位)
- **版本**: 最新版本 (推荐保持更新)

### 硬件配置
- **处理器**: 6 核或更多核心
- **内存**: 16GB RAM
- **存储**: SSD 硬盘 (提升文件处理速度)
- **显示**: 1920x1080 或更高分辨率

### 性能优化建议
- **多核心处理器**: 可以充分利用并行处理能力
- **SSD 存储**: 显著提升大文件处理速度
- **充足内存**: 减少临时文件交换，提升性能
- **稳定网络**: 快速下载 FFmpeg

- ❌ Windows RT (ARM 版本)

### 架构

- ❌ ARM 架构 (ARM64) Windows (未测试)

## 🔧 特殊功能要求

### NCM 解密功能
- **平台限制**: 仅在 Windows 平台支持
- **原因**: 使用 Windows DLL (`libncmdump.dll`)
- **其他平台**: 会显示错误提示，其他功能正常

### FFmpeg 支持
- **自动管理**: 程序会自动下载和配置 FFmpeg
- **手动安装**: 如自动下载失败，可手动下载 Windows 版本 FFmpeg
- **版本要求**: FFmpeg 4.0 或更高版本
- **离线使用**: 下载后可完全离线使用

### 便携版特性
- **无需安装**: 单文件便携版，直接运行
- **配置文件位置**: `%APPDATA%\zeus-music-maker\`
- **临时文件**: 系统临时目录，程序退出时自动清理
- **无注册表**: 不修改系统注册表

## 📊 性能参考

### 处理速度 (参考值)
- **音频转换**: 约 10-50MB/秒
- **视频转换**: 约 5-20MB/秒
- **并行处理**: 多文件同时处理，速度提升 2-4 倍
- **分片处理**: 大文件处理速度提升 3-5 倍

### 资源占用
- **基础内存**: 约 50-100MB
- **处理时内存**: 动态增长，取决于文件大小
- **CPU 使用**: 多核心并行，充分利用系统资源
- **磁盘 I/O**: 大缓冲区，减少系统调用

## 🐛 故障排除

### 常见问题

#### 1. 程序无法启动
- **检查**: 是否为 64 位 Windows 系统
- **检查**: 是否缺少 Visual C++ Redistributable
- **解决**: 从 Microsoft 官网下载并安装

#### 2. FFmpeg 下载失败
- **检查**: 网络连接是否正常
- **检查**: 防火墙是否阻止下载
- **解决**: 手动下载 FFmpeg 并放置到程序目录

#### 3. 字体显示异常
- **检查**: 系统是否安装了中文字体
- **解决**: 安装微软雅黑或其他中文字体

#### 4. 内存不足
- **检查**: 系统可用内存是否充足
- **解决**: 关闭其他程序，释放内存

#### 5. 处理速度慢
- **检查**: CPU 核心数是否足够
- **检查**: 是否为机械硬盘 (推荐 SSD)
- **解决**: 使用多核心处理器和 SSD 存储
